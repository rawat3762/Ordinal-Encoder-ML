# Ordinal-Encoder-ML

## 📌 Description
A hands-on Python project demonstrating Ordinal Encoding to preprocess categorical data and convert ordered categories into numerical values for machine learning models.

---

## 📌 What is Ordinal Encoding?
Ordinal Encoding is a technique used to convert categorical data (text values) into numerical values while maintaining their order.

---

## ⚙️ Workflow
- Import required libraries (pandas, scikit-learn) 
- Load dataset  
- Identify categorical features
- Apply OrdinalEncoder from Scikit-learn
- Convert ordered categories into numerical format
- Prepare dataset for machine learning models

---

## 🚀 Features
- Handles ordered categorical data efficiently
- Converts categories into meaningful numeric values
- Simple and clean preprocessing workflow
- Improves model compatibility with categorical data
- Beginner-friendly implementation

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- Scikit-learn (`OrdinalEncoder`)  
- Jupyter Notebook
